# Projet de Fin d'Étude

## Description
Application web développée en PHP avec HTML, CSS et JavaScript.

## Fonctionnalités
- Gestion des utilisateurs
- Interface web dynamique
- Connexion à une base de données MySQL

## Technologies
- PHP
- MySQL
- HTML / CSS / JavaScript

## Installation

1. Cloner le projet :
git clone https://github.com/username/projet.git

2. Importer la base de données :
database/projet.sql

3. Configurer la connexion :
modifier includes/config.php

4. Lancer avec XAMPP :
http://localhost/projet
